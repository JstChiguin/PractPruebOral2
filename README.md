# Práctica Oral 2

Aplicación web creada para practicar evaluaciones orales de inglés.

## Características

* Conversaciones guiadas para practicar preguntas y respuestas.
* Pronunciación escrita para facilitar el estudio.
* Audio utilizando la voz del navegador.
* Ejercicios de predicción con "Going To".
* Ejemplos de Simple Past y Present Perfect.
* Casos prácticos con personajes conocidos como Lionel Messi y Cristiano Ronaldo.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Objetivo

Este proyecto fue desarrollado como apoyo para la preparación de pruebas orales de inglés nivel B1, permitiendo practicar pronunciación, comprensión y estructuras gramaticales de forma interactiva.

## Cómo usar

1. Abrir la aplicación en el navegador.
2. Seleccionar la sección que se desea practicar.
3. Presionar el botón "Escuchar".
4. Repetir las frases en voz alta.
5. Utilizar "Escuchar todo seguido" para realizar prácticas completas.
